# 100DaysTracker App

## Installation
You need to install [SQLite](https://www.sqlite.org/index.html) for database.
<br>
Then on each `root` and `client` directory:
```bash
yarn install
```

<br>

At root directory:
#### Create database:
```bash
yarn createdb
```

#### Start the server (port 5000)
```bash
yarn dev
```

#### Start the client (port 3000)
```bash
yarn client
```
